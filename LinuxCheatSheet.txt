Linux Command Cheat Sheet;

ls - List all major files and directories in a given file system

cd - Change file directory
 augmentations: .. up one directory in chain 
		./add to current directory chain

mv - Move a fiel from one folder or directory to another 

man - displays the manual of the input command 

mkdir - make directory - allows the user to create a new directory (empty directory)

rmdir - removes a directory (only empty directory)
